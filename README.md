# B.Tech Project(BTP)

## Name
Smart Search Engine to Handle Phrase Queries

## Description
This project implements a search engine using Elastic Search Framework for IIT Palakkad Institute website which supports features like
- Keyword Search
- Phrase Queries
- Quotes Queries
- Image Queries
